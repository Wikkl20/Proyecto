Introducción a la Semana 6

En esta semana desarrollamos la aplicación Agua Clara utilizando Java y MySQL con la arquitectura MVC. El proyecto se centra en la gestión de datos de clientes, incluyendo operaciones de registro, actualización, eliminación y consulta, implementadas mediante una base de datos relacional y una interfaz gráfica.

Actividad 01: Gestión de Clientes

La aplicación Agua Clara permite gestionar información de clientes, incluyendo su nombre, dirección, teléfono y consumo de agua. Se implementó una base de datos en MySQL para almacenar los datos y operaciones CRUD (Crear, Leer, Actualizar, Eliminar) utilizando el patrón MVC con paquetes modelo, vista y controlador. 


package modelo;

public class Cliente {
    private int id;
    private String nombre;
    private String direccion;
    private String telefono;
    private double consumo;

    public Cliente(int id, String nombre, String direccion, String telefono, double consumo) {
        this.id = id;
        this.nombre = nombre;
        this.direccion = direccion;
        this.telefono = telefono;
        this.consumo = consumo;
    }

    // Getters y Setters
    public int getId() { return id; }
    public void setId(int id) { this.id = id; }
    public String getNombre() { return nombre; }
    public void setNombre(String nombre) { this.nombre = nombre; }
    public String getDireccion() { return direccion; }
    public void setDireccion(String direccion) { this.direccion = direccion; }
    public String getTelefono() { return telefono; }
    public void setTelefono(String telefono) { this.telefono = telefono; }
    public double getConsumo() { return consumo; }
    public void setConsumo(double consumo) { this.consumo = consumo; }
}

La vista muestra formularios para ingresar y consultar datos, mientras el controlador gestiona la lógica y la conexión con la base de datos.

Actividad 02: Conexión a Base de Datos

Se implementó una conexión a una base de datos MySQL para realizar operaciones CRUD en la tabla de clientes. La clase de conexión utiliza JDBC para interactuar con la base de datos, permitiendo registrar, actualizar, eliminar y consultar información de los clientes de manera eficiente. 


package controlador;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class Conexion {
    private Connection conn;
    private String url = "jdbc:mysql://localhost:3306/agua_clara";
    private String user = "root";
    private String pass = "";

    public Connection conectar() {
        try {
            conn = DriverManager.getConnection(url, user, pass);
            System.out.println("Conexión exitosa");
            return conn;
        } catch (SQLException e) {
            System.out.println("Error de conexión: " + e.getMessage());
            return null;
        }
    }

    public void desconectar() {
        try {
            if (conn != null) {
                conn.close();
                System.out.println("Conexión cerrada");
            }
        } catch (SQLException e) {
            System.out.println("Error al cerrar conexión: " + e.getMessage());
        }
    }
}

El controlador utiliza esta clase para gestionar todas las operaciones de la base de datos, asegurando una interacción fluida entre la interfaz y los datos.
