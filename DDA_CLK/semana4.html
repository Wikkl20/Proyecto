Introducción a la Semana 4

En esta semana trabajamos con Java y Apache NetBeans usando arquitectura MVC. Los proyectos se enfocan en cálculos geométricos y validación de usuarios con niveles de acceso, además de interfaces gráficas.

Actividad 01: Figuras geométricas

Se desarrolla una app que permita ingresar dimensiones de cuadrado, rectángulo y triángulo. La aplicación calcula el área y el perímetro, y dibuja la figura en pixeles. Todo bajo el patrón MVC con paquetes modelo, vista y controlador. 


// Ejemplo de clase Modelo (Cuadrado)
package modelo;

public class Cuadrado {
    private double lado;

    public Cuadrado(double lado) { this.lado = lado; }
    public double calcularArea() { return lado * lado; }
    public double calcularPerimetro() { return 4 * lado; }
}

La vista solicita datos y muestra resultados, mientras el controlador gestiona la interacción.

Actividad 02: Login y conversiones

Una aplicación que incluye un login para validar dos o más usuarios. Cada usuario tiene un nivel de acceso distinto. En la segunda ventana se implementa un sistema de conversiones de cuatro medidas (ejemplo: metros a centímetros, kilómetros a millas, etc.). 


// Ejemplo básico de login (Modelo Usuario)
package modelo;

public class Usuario {
    private String username;
    private String password;
    private String rol;

    public Usuario(String username, String password, String rol) {
        this.username = username;
        this.password = password;
        this.rol = rol;
    }

    public boolean validar(String u, String p) {
        return username.equals(u) && password.equals(p);
    }

    public String getRol() { return rol; }
}

La lógica de validación se implementa en el controlador y las conversiones en la segunda ventana.
