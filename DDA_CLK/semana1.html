Introducción a la Semana 1

¡Cada semana que avanzas, estás más cerca de ser un gran programador!

En esta primera semana nos introducimos al uso de interfaces gráficas en Java utilizando Apache NetBeans. Los ejercicios desarrollados nos permiten conectar la lógica de programación con interfaces interactivas, aplicando validaciones, cálculos y presentación clara de resultados.

Problema 1: Tienda de Camisas con Descuento 7% + 7%

Una tienda ha puesto en oferta la venta de camisas con un descuento especial por temporada de verano, denominado 7% + 7%. Los cálculos deben realizarse de la siguiente forma:

  • El importe de la compra es el precio de la camisa multiplicado por la cantidad adquirida.
  • El primer descuento es el 7% del importe de la compra.
  • El segundo descuento es el 7% de lo que queda después del primer descuento.
  • El descuento total es la suma de ambos descuentos.
  • El importe a pagar es igual al importe inicial menos el descuento total.

Diagrama de Clases — Problema 1


class Camisa {
  - precio: double
  - cantidad: int
  + calcularImporte(): double
}

class Descuento {
  - porcentaje: double
  + aplicar(): double
}

Camisa "1" *-- "1" Descuento

El diseño muestra cómo una Camisa se relaciona con un Descuento. La clase Camisa calcula el importe, mientras que Descuento aplica el porcentaje.

Problema 2: Presupuesto de Inversión en Feria

Una empresa participará en una feria y debe distribuir un monto total de inversión en distintos rubros. El programa debe calcular cuánto corresponde a cada rubro según los porcentajes establecidos:

Rubro Porcentaje
Alquiler de espacio en la feria23%
Publicidad7%
Transporte26%
Servicios feriales12%
Decoración21%
Gastos varios11%

El programa recibe el monto total de inversión y devuelve cuánto dinero corresponde a cada rubro.

Diagrama de Clases — Problema 2



class Presupuesto {
  - montoTotal: double
  + distribuir(): void
}

class Rubro {
  - nombre: String
  - porcentaje: double
  + calcularMonto(): double
}

Presupuesto "1" *-- "many" Rubro

Aquí la clase Presupuesto administra el monto total y lo distribuye en varios Rubros (espacio, publicidad, transporte, etc.), cada uno con su porcentaje definido.
